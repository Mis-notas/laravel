## Instalar certbot
~~~TERM
sudo snap install --classic certbot
~~~
Ahora se configura dependiendo del servidor que se esté usando (nginx - apache)
~~~TERM
sudo certbot --nginx -d name.com example.com
~~~
~~~TERM
sudo certbot --nginx 
~~~
Si no elegimos el dominio con la opción -d certbot nos dara la opción de elegir cual dominio es el que queremos certificar.

Escribimos un email que nos pide el sistema ->
Y aceptamos las políticas ->
y debemos seleccionar que force que salga siempre por https (a veces no sale esta opción)
Finalmente nos entrega los ssh

## Renovación
Este proceso tiene un duración de 3 meses, por lo que se debe realizar un renovación, sin embargo certbot brinda la siguiente opción 
~~~PHP
sudo certbot renew --dry-run
~~~
