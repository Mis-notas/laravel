## Instalar y seleccionar diferentes versiones de PHP
~~~TERM
sudo apt update
~~~
~~~TERM
sudo apt install php7.4
~~~
Para escoger la versión que se quiere utilizar 
~~~TERM
sudo update-alternatives --set php /usr/bin/php7.4
~~~
Ahora para decirle a Apache que use una determinada versión desactivamos la versión actual escribimos:
~~~TERM
sudo a2dismod php7.2
~~~
Luego de ésto debemos habilitar la versión de PHP que deseemos:
~~~TERM
sudo a2enmod php7.4
~~~
~~~TERM
sudo systemctl restart apache2
~~~

## Habilitar firewall para https
~~~TERM

~~~
~~~TERM
sudo ufw status
~~~
